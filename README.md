# Ingredient Search Site

## Описание проекта

**Ingredient Search Site** — веб-приложение для поиска рецептов на основе набора ингредиентов, введённых пользователем. Пользователь последовательно добавляет продукты, после чего приложение анализирует доступную базу рецептов и формирует список блюд, содержащих указанные ингредиенты. Для каждого найденного рецепта отображается ссылка на источник, а также информация о пищевой ценности.

Проект ориентирован на наглядную демонстрацию работы серверной логики обработки данных и клиентского интерфейса веб-приложения.

---

## Архитектура приложения

### Серверная часть

Серверная логика реализована на языке Python.  
Основной файл проекта — `main.py`, который:

- принимает запросы от клиентской части;
- проверяет наличие введённых ингредиентов в базе данных;
- формирует и возвращает результаты поиска в формате JSON.

Файл `class_names.json` используется для хранения вспомогательных данных, необходимых для корректной работы серверной логики.

---

### Клиентская часть

Клиентский интерфейс реализован с использованием HTML, CSS и JavaScript.

- `templates/index.html` — основной шаблон веб-приложения, содержащий форму ввода ингредиентов, элементы управления и клиентскую логику взаимодействия с сервером.
- `templates/info.html` — дополнительный информационный шаблон.

Оформление интерфейса выполнено с поддержкой двух тем:

- `static/lightTheme.css` — светлая тема;
- `static/darkTheme.css` — тёмная тема.

Переключение тем осуществляется динамически на стороне клиента.  
Графические элементы интерфейса размещены в каталоге `static/images`, включая файл `burger.png`.

---

## Данные

В качестве источника информации для формирования базы рецептов используются CSV-файлы:

- `povarenok_recipes_2021_06_16.csv`
- `povarenok_recipes_cleaned.csv`
- `recipes_with_ingredients_and_quantities.csv`

Данные файлы не включены в репозиторий из-за большого объёма, однако именно они используются для извлечения основной информации, применяемой в работе веб-приложения.
